# **App Name**: CallSync Notes

## Core Features:

- Call Log Sync: Automatically sync call history from the Android device, with correct permissions being requested.
- Call Log Display: Display the comprehensive call log in a clean, organized manner.
- Contact Name Resolution: Resolve contact names for phone numbers in the call log using Android contacts.
- Call Note Entry: Allow users to add specific notes to individual call entries.
- Persistent User Notes: Attach persistent notes to phone numbers, retrievable across call entries.

## Style Guidelines:

- Background color: Light blue (#E0F7FA) for a clean, modern feel.
- Primary color: Blue (#2979FF) for primary interactions and key elements.
- Accent color: Slightly desaturated light blue (#53A3FF) for highlighting new or important information.
- Body and headline font: 'Inter', a sans-serif font for a modern, clean interface.
- Use simple, line-based icons in shades of blue and gray to maintain a consistent aesthetic.
- Employ a card-based layout for call entries, with clear separation of information like contact name, number, date, and notes.
- Incorporate subtle animations for transitions and interactions, such as sliding in new call entries or expanding note fields.
