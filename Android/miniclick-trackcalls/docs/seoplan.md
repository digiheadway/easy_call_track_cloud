## ğŸ›  Technical SEO Checklist

### On-Page SEO

- [ ] **Title Tags**: Unique, 50-60 characters, primary keyword included
- [ ] **Meta Descriptions**: Unique, 150-160 characters, CTA included
- [ ] **H1 Tags**: One per page, includes target keyword
- [ ] **H2-H6 Hierarchy**: Proper nesting, keyword-rich
- [ ] **Image Alt Text**: Descriptive, includes keywords where natural
- [ ] **URL Structure**: Short, lowercase, hyphens, no special characters
- [ ] **Internal Links**: 3-5 per page minimum
- [ ] **External Links**: 1-2 authoritative sources per content page
- [ ] **Schema Markup**: Product, FAQ, HowTo, Organization schemas

### Technical Infrastructure

- [ ] **Mobile Responsiveness**: All pages mobile-optimized
- [ ] **Page Speed**: <3s load time (Test with PageSpeed Insights)
- [ ] **Core Web Vitals**: LCP <2.5s, FID <100ms, CLS <0.1
- [ ] **HTTPS**: SSL certificate active
- [ ] **XML Sitemap**: Generated and submitted to GSC
- [ ] **Robots.txt**: Properly configured
- [ ] **Canonical Tags**: On all pages
- [ ] **Hreflang**: If multi-language in future
- [ ] **404 Page**: Custom branded 404

### Analytics & Tracking

- [ ] **Google Analytics 4**: All pages instrumented
- [ ] **Google Search Console**: Verified and monitored
- [ ] **Conversion Tracking**: Signup, Download, Contact events
- [ ] **Heatmaps**: Hotjar or similar for UX insights
