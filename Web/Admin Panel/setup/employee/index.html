Employee Setup Guide

Follow these steps to link your device and start tracking your work calls.

1

Download the App

Get "Call Track & Manage by MiniClick" from the Play Store.

Download for Android
2

Enter Your Login Code

Use the code or credentials provided by your manager.

Example Code Format:

XC-9921

3

Enable System Call Recording

The app syncs recordings from your phone's built-in recorder. You must enable it in your phone settings.

Open Phone App → Settings → Record Calls → Set to "Auto Record" or "All Calls".

4

Grant Permissions

Allow the app to access necessary features. This is required for call tracking and recording sync.

Phone Calls
Call Logs
Files & Media (Storage)
Contacts
5

Disable Battery Optimization

This is critical to ensure the app runs in the background.

Important: Go to Settings → Apps → Call Manage → Battery → Set to "Unrestricted". This prevents Android from killing the app.

6

You're All Set!

The app will now automatically track and sync your calls. Look for the "MiniClick is active" notification.

Tracking is now active

Troubleshooting

Recordings are not syncing

Make sure the device's built-in call recorder is enabled and the app has Storage permission to access those files.

Calls are not syncing to cloud

Check your internet connection. If "Wi-Fi Only" mode is ON, connect to Wi-Fi to sync.

App keeps stopping

Xiaomi, Samsung, and Oppo phones have aggressive battery savers. Enable "Autostart" in phone settings.
